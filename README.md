  
**Web 3: Unit 2 Project**  
**EVIL FICTIONAL CHARACTER**

**![Darth Vader](./src/images/darth.png)**

**Overview:** You have been hired to create a website for the evil fictional character of your choice.  Plan and execute a great site using **CSS Grid & Flexbox** while using **VS Code** ...and remember...you’d hate to disappoint a villain\! 

**Examples:**

* [Example 1](https://alexcovar.github.io/evilWebsite/index.html)  
* [Example 2](https://im-not-okay.github.io/evilFictionalCharacter/index.html)  
* [Example 3](https://naixisle.github.io/evilFictionalCharacter/)

**Requirements:**

* Site must have 3 pages  
  * Mandatory \- An **index.html** landing page  
  * Ideas for pages you can create:  
    * A store \- where the villain sells their merch as a side hustle ([example](http://www.oafe.net/shocka/swtater.php))  
    * A contact form  
    * An about page  
    * A media page with videos  
    * A gallery page with photos  
* Float, Clear, CSS Grid & Flexbox must be applied on all pages  
* Google Fonts must be implemented throughout the site  
* Links must be present on one or more pages  
* Semantic HTML tag structure (\<header\>, \<main\>, \<nav\>, \<figure\>, etc…)  
* **One** video and **one** audio  
* Note: Text may be copied from the internet\!
